# 🚀 Python User Management API - Continuous Integration Pipeline

A beginner-friendly DevOps project demonstrating how to build a Continuous Integration (CI) pipeline using **Jenkins**, **GitHub**, **Flask**, and **pytest**.

The project includes a simple Flask API, automated testing with pytest, and a Jenkins Pipeline that automatically validates the application whenever new code is pushed to GitHub using GitHub Webhooks.

This project was built as part of a DevOps learning roadmap to understand Continuous Integration fundamentals before moving on to Docker, Kubernetes, Terraform, and AWS.
